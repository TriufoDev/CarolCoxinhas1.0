# Carol Coxinhas — Site

Projeto estático para apresentar e gerenciar pedidos simples de coxinhas.

## Visão geral

Site estático contendo páginas para catálogo, administração e acompanhamento de pedidos.

## Como visualizar localmente

- Abra a página principal no seu navegador: [index.html](index.html)
- Para servir localmente (opcional):

```bash
python -m http.server 8000
# então abra http://localhost:8000
```

## Estrutura do projeto

- [index.html](index.html) — página principal
- [admin.html](admin.html) — painel administrativo
- [pedidos.html](pedidos.html) — lista de pedidos
- [acompanhar-pedido.html](acompanhar-pedido.html) — acompanhamento de pedido
- [carolCoxinhas.css](carolCoxinhas.css) — estilos específicos do projeto
- [styles.css](styles.css) — estilos globais
- [script.js](script.js) — comportamento em JavaScript

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Desenvolvimento

- Faça alterações nos arquivos acima e teste abrindo `index.html` ou usando um servidor local.
- Para contributos: crie uma branch, faça commits descritivos e envie um pull request.

## Observações

- Este repositório contém apenas front-end estático. Se precisar de persistência de pedidos, integre um back-end ou use serviços externos.
